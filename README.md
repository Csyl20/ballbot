# Code for a self balancing ballbot
WORK IN PROGRESS

Hardware specs:
- 3x Maxon DCX22s motors with ENX16 Easy 64CPR encoders
- 3x Omniwheels
- 1x Teensy 3.1 @ 96MHz
- 1x Invensense MPU9250 9dof IMU
- 3d printed and laser cut custom parts
